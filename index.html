⚙️ The Core Commands of Filesystem Navigation

pwd Print Working Directory

Shows your current exact location inside the system's nested directories, stating the absolute directory path starting from root (`/`).

ls List Directory Contents

Reveals all file and sub-directory names residing inside your current directory. Essential for identifying configuration logs or hidden keys.

cd [target] Change Directory

Moves you to another folder path. Type cd .. to step back up one folder layer, or cd to return straight home.

cat [file] Concatenate & Display

Renders and outputs the raw text stored within a file directly into the terminal window—used to inspect critical scripts and credential files.

Essential Commands
